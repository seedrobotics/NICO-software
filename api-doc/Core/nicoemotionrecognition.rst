nicoemotionrecognition
**********************

The **nicoemotionrecognition** package contains a model to recognize and reaspond
to emotions from face images.

EmotionRecognition
##################

The EmotionRecognition class detects faces from camera images and recognizes
the displayed emotion. It can also mirror the emotion and give voice feedback
based on it as well as follow the detected face with the robots head.
